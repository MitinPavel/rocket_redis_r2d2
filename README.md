# Redis + r2d2 connection pool + Rocket.rs

A simple example app for:
* [Rocket web framework](https://rocket.rs/)
* [Redis db](https://redis.io/)
* [r2d2 connection pool](https://github.com/sfackler/r2d2)
## Tags

Rust, Rocket, Redis, r2d2
 
## Getting started

```
cargo run
```
